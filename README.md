# Horizon - Student Analytics Platform

A Django-based Student Analytics Platform that transforms raw school ERP data into meaningful academic insights for Parents, Teachers, and School Management.

---

## Features

- Parent Analytics Dashboard
- Student Academic Performance
- Attendance Analytics
- Subject-wise Performance Analysis
- Subject Growth Journey
- Library Analytics
- Project Tracking
- Achievement Tracking
- Remedial Learning Analysis
- AI Generated Student Summary
- Multi Academic Year Comparison
- Secure Role-based Login

# Installation

```bash
git clone https://github.com/<username>/horizon.git

cd horizon

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```
